## ⚙️ Installation & Setup

1. **Clone the repository**

   ```bash
   git clone https://github.com/your-username/chatbot-app.git
   cd chatbot-app
   ```

2. **Install dependencies**

   * Backend

     ```bash
     cd backend
     npm install
     ```
   * Frontend

     ```bash
     cd ../frontend
     npm install
     ```

3. **Start the servers**

   * Backend

     ```bash
     cd backend
     npm start
     ```
   * Frontend

     ```bash
     cd frontend
     npm start
     ```
